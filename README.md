# Zebedee Realty - Premium Institutional Construction Website

A modern, high-end, contemporary luxury website for Zebedee Realty, the execution arm of Zebedee Ventures. Built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Contemporary Luxury Design**: Styled with a sophisticated Deep Forest Green (`#1a3c34`), Gold (`#cba052`), and Sand (`#f5f5f0`) color palette.
- **Premium Typography**: Features `Marcellus` for elegant serif headings and `Roboto` for clean, readable body text.
- **Cinematic Hero**: Uses an immersive 8-second 16:9 8k video background showcasing luxury villa development.
- **Flagship Showcase**: Highlights the 100-villa Serene Canterbury institutional project for the Manipal Group.
- **Services Architecture**: Clearly structured service cards for Civil Construction, Joint Development Execution, and High-spec Finishing.
- **SEO Optimized**: Complete with metadata, Open Graph tags, Twitter cards, and JSON-LD structured data.

## Technology Stack

- Vite
- React 19
- TypeScript
- Tailwind CSS v3
- Framer Motion
- Lucide React
- React Router DOM

## Getting Started

1. Ensure dependencies are installed: `pnpm install`
2. Run the development server: `pnpm dev`
3. Build for production: `pnpm build`
4. Lint and check types: `pnpm run typecheck && pnpm run lint`

## Project Structure

- `src/components/layout`: Contains global wrappers (`Layout.tsx`, `Navigation.tsx`, `Footer.tsx`).
- `src/pages`: Individual route pages (e.g., `Home.tsx`).
- `src/routes`: Configuration for `react-router-dom`.
- `src/index.css`: Global CSS containing HSL color variables and Tailwind base imports.
